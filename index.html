Check out who Perez liked and disliked the most at this year's Grammys!








How was this done?

I love looking at pop culture, so I put together this site that goes through all of Perez Hilton's Tweets about The Grammys. I used the Twitter API within R to search for all of the people that were mentioned or tagged in Perez's comments. I then used a sentiment analysis function in R to assess how positive or negative these statements were for each person. Most people received scores of 0, which means that the comments were neutral about them. The ones left in the chart display how negative or positive Perez was towards them during this award season. A score above 0 is positive and below is negative. I created the Wordcloud using the wordcloud package in python and a picture of Perez that I found online.
